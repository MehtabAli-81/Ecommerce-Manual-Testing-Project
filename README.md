🛒 E-Commerce Manual Testing Project
📌 Project Overview
This project demonstrates manual testing of an E-commerce website.
The goal is to ensure that all major functionalities like login, signup, product search, cart, and checkout are working correctly.
________________________________________
🎯 Objectives
•	To validate core functionalities of the application
•	To identify bugs and ensure quality
•	To create proper testing documentation
________________________________________
 Testing Scope
•	Login & Signup
•	Product Search
•	Add to Cart
•	Checkout Process
•	User Profile
•	Logout
________________________________________
🛠 Testing Type
•	Functional Testing
•	UI Testing
